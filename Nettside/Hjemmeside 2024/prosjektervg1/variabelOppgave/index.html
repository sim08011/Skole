Variabler

Om variabler:

En variabel er en tilordnet verdi som inneholder en eller annen kjent eller ukjent mengde av informasjon.

Dette er veldig praktisk hvis du skal bruke den samme verdien flere plasser i koden din, og også endre på dens verdi mens du jobber, eller mens siden kjører.


Hvordan deklarere en variabel i JavaScript.

Du kan bruke enten "let" eller "var", men det anbefales å bruke let, på grunn av at det er den nye standaren.

Da skriver du sånn: "let x = y" (x er navnet på variabelen og y er verdien du tilordner).


Eksempel: Addering av to ulike variabler og gi summen til en annen variabel.

script.js

let tall1 = 5; // Deklarerer variablen tall1 og tilordner den verdien 5

let tall2 = 8; // Deklarerer variablen tall2 og tilordner den verdien 8

let sum = 0; // Deklarerer variabelen sum og tilordner den verdien 0

sum = tall1 + tall2; // Legger sammen variablene og gir verdien til sum variablen

console.log(sum); // Skriver ut summen i konsollen.


Sjekk gjerne konsollen for flere eksempler. Dette gjør du ved å høyreklikke et sted på nettsiden og trykke "inspiser" eller "undersøk" og deretter trykker på "konsoll" fanen.

Navn og operatører

Navn

Man bør alltid gi variablene sine beskrivende navn, så man skjønner hva de er til. Du bør også bruke camelcase. Finn ut mer.

Operatører

Man regner ofte med variablene sine, derfor er det sjekt å kunne de forskjellige operatørerene.

= Oppdragsoperatør a = 7; // 7
+= Tilleggsoppgave a += 5; // a = a + 5
-= Subtraheringsoppgave a -= 2; // a = a - 2
*= Multiplikasjonsoppgave a *= 3; // a = a * 3
/= Divisjonsoppgave a /= 2; // a = a / 2
%= Resterende oppgave a %= 2; // a = a % 2
**= Eksponentieringsoppgave a **= 2; // a = a**2
